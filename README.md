# number-guessing-game
a number guesing game that uses binary search for learning purposes

## Showcase
```
==================================================
Welcome to the number guessing game!
==================================================
This game was created to show you how optimal binary search can be

Think of a number between 0 and 100.

When you're ready, type 'Y' to begin.

Have you picked a number? (Y/N): y
Let's begin!

Is your number 50? (y/n): n
Hmm, interesting... Is your number higher or lower than 50? (h/l): l

Is your number 25? (y/n): n
Hmm, interesting... Is your number higher or lower than 25? (h/l): l

Is your number 12? (y/n): n
Hmm, interesting... Is your number higher or lower than 12? (h/l): l

Is your number 6? (y/n): y
Yay! I got your number in 4 attempts! Have a nice day!
```

## Getting Started
```bash
$ git clone htts://github.com/ibnaleem/number-guessing-game.git
$ cd number-guessing-game
$ python3 game.py
```
